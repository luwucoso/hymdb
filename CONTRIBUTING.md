## Adding a new mod

Adding a new mod is very straight-forward.

If you are comfortable with Git lingo, just add your file into `mods/ORG/MOD.json` and make a PR. Check out the *Writing The File* section for the required fields, or just copy `mods/example/tiny_mod.json` into the appropriate directory and modify it.

If not, keep reading!
